# 盯梢 — 找用户计划

## 目标用户画像
1. **产品经理/运营** — 需要监控竞品官网变化（功能更新、定价调整）
2. **独立开发者** — 关注 GitHub Trending、HN、ProductHunt 上的新项目
3. **电商从业者** — 监控竞品价格、促销信息
4. **求职者** — 盯着目标公司招聘页有没有新岗位
5. **研究员/记者** — 追踪政府网站、行业报告更新

## 差异化卖点
- **零部署** — 不需要 Docker/服务器，注册即用
- **AI 摘要** — 不只告诉你"变了"，用 AI 告诉你"变了什么"
- **中文优先** — 界面、通知、摘要全中文
- **飞书/微信推送** — 变化直接推到你用的 IM

## 第一波推广渠道（不需要 X 参与）

2. 知乎"有什么好用的网页监控工具"问题下回答
3. 小红书笔记（需要 X 发布 — 对外内容需审核）
4. GitHub 开源（后续考虑）

- 本周找到 5 个真实用户注册使用
